# 3P Solutions Website

Live Next.js website for 3P Solutions / Brent Pattison, deployed through Vercel.

## Framework

- Next.js App Router
- React
- Tailwind CSS
- Vercel deployment from GitHub

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```
